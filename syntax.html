Syntax Cheat Sheet

Våra element:

  • p-taggen (p): den används för att tala om att här kommer det ett stycke text.
  • unordered list (ul): den används för att markera att en lista startar här.
  • li-taggen (li): den används i antingen en ol (ordered list) eller en ul (unordered list) för att få punkterna i listan
